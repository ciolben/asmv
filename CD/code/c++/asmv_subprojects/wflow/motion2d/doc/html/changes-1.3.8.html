Change notes for Motion2D 1.3.8



This file describes the changes between Motion2D 1.3.7 and 1.3.8.

General

  • Possibility to get the covariance matrix of the estimated motion model parameters.
  • Bug fixes.

Library

  • CMotion2DEstimator: Addition of new methods to get the covariance matrix of the estimated motion model parameters:
    • CMotion2DEstimator::computeCovarianceMatrix()
    • CMotion2DEstimator::isCovarianceMatrixComputed()
    • CMotion2DEstimator::getCovarianceMatrix()

  • CMotion2DEstimator: The method CMotion2DEstimator::initialize() was removed (it becomes obsolete).

  • CMotion2DPyramid: Suppression of memory leeks when images are very small.

Examples

  • The Motion2D example:
    • Addition of an option (-z) to compute and print the covariance matrix of the estimated motion model parameters.

Documentation

  • Some improvements.
