This folder contains handwritten functions for Matlab.

analyzeclust	: directly plot motion_t values as raw, smoothed with ws 15, smoothed with ws 24, and save as png file the plot.
movingavg	: my implementation of the running average for tests.
readcsv		: parse the output of Particle Video and transform it into an array of cells (required by CCToolbox). Comment/uncomment some lines to change its behavior.

RESULT folder.
Contains some interesting outputs.

CCToolbox folder.
Contains the CCToolbox tool for clustering the curves, as I used it. Read the README inside the folder.
