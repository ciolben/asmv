Change notes for Motion2D 1.3.6



This file describes the changes between Motion2D 1.3.5 and 1.3.6.

General

  • This version is the first release created specificaly for an open-source distribution as soon as possible.
  • Addition of some specific purpose for this open source release (license, configuration script, copyright, ...)
  • Multi platform support (SunOS, Solaris, Linux, IRIX64, Windows)

Library

  • Bugfixes and improvements
  • Available API:
    • CMotion2DEstimator: Robust multiresolution estimation of parametric motion models.
    • CMotion2DImage: Basic image manipulations.
    • CMotion2DModel: Parametric motion models specification.
    • CMotion2DPyramid: Gaussian image and spatial gradients pyramids manipulation.
    • CMotion2DPyramid: Motion compensation.

Examples

  • Addition of some examples to show how to use the Motion2D library.

Documentation

  • Creation of a Motion2D dedicated web site: http://www.irisa.fr/vista/Motion2D
  • API documentation in english.
  • online documentation built with doxygen.
